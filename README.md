# TaskBite

Official releases and installer downloads for TaskBite, a desktop task manager
with reminders, focus timers, categories, and optional encrypted cloud sync.

- [TaskBite website](https://taskbiteapp.com)
- [Latest release](https://github.com/simbrr/TaskBite/releases/latest)
- [Report a problem](https://github.com/simbrr/TaskBite/issues)

The application source code is maintained in a private repository. This public
repository contains distribution metadata and release artifacts only.
